# Kubernetes Manifests

This folder is where you'll create your Kubernetes manifests for the DevOps System.

## Level 2 - Kubernetes

Create the following files here:
- `client-deployment.yaml` - Deployment for the React client
- `client-service.yaml` - Service to expose the client
- `server-deployment.yaml` - Deployment for the Strapi server
- `server-service.yaml` - Service to expose the server
- `ingress.yaml` (optional) - Ingress for routing

Refer to `devops-kit/level-2/helper/k8s/` for complete reference implementations.
