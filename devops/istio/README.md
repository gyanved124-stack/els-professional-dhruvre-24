# Istio Service Mesh

This folder is where you'll add Istio configurations for advanced traffic management.

## Level 5 - Advanced DevOps

Add the following configurations here:
- `virtual-service.yaml` - Traffic routing rules
- `destination-rule.yaml` - Load balancing and circuit breaker policies
- `gateway.yaml` - Ingress gateway configuration

**Prerequisites:**
- Istio should be installed
- Namespace should have istio-injection enabled

Refer to `devops-kit/level-5/helper/` for reference configurations.
